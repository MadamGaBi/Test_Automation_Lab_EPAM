Лабораторная работа 4
Создание простого класса, контроллера для ввода данных с командной строки с помощью регулярных выражений 
-написать класс согласно самостоятельно созданному варианту даннях в классе, оформить согласно JCC,  написать JavaDOC;

и в случае полной валидности всех данных – передает их в соответствующий класс.
